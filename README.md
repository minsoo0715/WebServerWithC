## Concurrent Web Server using BSD Sockets

### Developement Env.
```text
make - GNU Make 4.3 (Built for x86_64-pc-linux-gnu)
gcc  - gcc version 11.4.0 (Ubuntu 11.4.0-1ubuntu1~22.04)
```

### Build
```bash
$ make
```

### Run
```bash
$ ./main.o {portNumber}
$ make && ./main.o {portNumber} # With build
```
